this application is written in php . so you need php runtime installed on your system.

[Running on windows]

install xampp from https://www.apachefriends.org/download.html
install it in the default location ex. c:\xampp

now go to extracted folder and find the file windows-scraper.bat
and run that batch file by douuble clicking it.

[Running on mac]


install brew from here 
https://brew.sh/

brew update 
brew upgrade
brew install php@7.4
brew link php@7.4 

[linux]

for linux user you use your package manager to install php . and from the terminal you can run 
php main.php 
to execute the scraper .
